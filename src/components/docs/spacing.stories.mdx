# Spacing

The spacing scale is defined as follows:

| Naming | Size in Pixel |
| ------ | ------------- | ------------------------------------- |
| xxs    | 4             | <div className="spacing w-xxs"></div> |
| xs     | 8             | <div className="spacing w-xs"></div>  |
| s      | 16            | <div className="spacing w-s"></div>   |
| m      | 24            | <div className="spacing w-m"></div>   |
| l      | 32            | <div className="spacing w-l"></div>   |
| xl     | 48            | <div className="spacing w-xl"></div>  |
| xxl    | 64            | <div className="spacing w-xxl"></div> |

The spacing scale is inherited by the padding, margin, width, height, maxHeight, gap, inset, space, and translate core plugins of tailwind.
