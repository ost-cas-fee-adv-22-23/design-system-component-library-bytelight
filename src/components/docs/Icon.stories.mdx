## Usage

Each icon is a single react component. All you have to do is to import the icon.

```jsx
import { MumbleIcon } from '@smartive-education/design-system-component-library-bytelight';
```

### Color and Size

The color of the icon inherits the current text color of the parent element. If required, you can easily assign a new color to the icon via a wrapper. In the following example the color is set on the parent div element of the icon. The size of the icon could be set via the size prop if no size is set, the size will be 100%.

```jsx
<div className="text-slate-700">
  <IconMumble size="16" />
</div>
```
